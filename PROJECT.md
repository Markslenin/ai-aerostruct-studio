# 项目核心说明

## 项目名称

AI AeroStruct Studio

## 项目目标

构建一套面向低速飞行器翼面/弹翼初步设计的轻量化智能辅助设计平台。系统采用 LLM + Skills 架构，大语言模型只负责需求理解、知识检索、任务调度和结果解释，关键工程计算由确定性 Skill 模块完成。

## 核心思想

```text
LLM 负责理解和组织
Skill 负责计算和校核
规则模块负责最终判定
结果存储负责可追溯
```

## 目标闭环

```text
用户需求
↓
DesignSpec
↓
几何生成
↓
气动计算
↓
结构校核
↓
规则判定
↓
报告输出
```

## 适用范围

第一阶段限定为：

- 低速飞行器翼面；
- 梯形翼；
- 简化翼型；
- 初步气动估算；
- 简化悬臂梁结构校核；
- 设计报告生成。

## 非目标

短期内不追求：

- 完整 CAD 几何内核；
- 高精度 CFD；
- 完整三维有限元；
- 自动替代工程师判断；
- 直接闭环控制真实飞行器设计。

## 架构优势

1. 降低 LLM 幻觉风险；
2. 工程计算可追溯；
3. 各模块可独立测试；
4. 后续可逐步替换为 C/C++ 高性能内核；
5. 适合作为毕业设计/科研 Demo/工程原型。
